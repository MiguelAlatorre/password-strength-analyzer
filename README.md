# Password Strength Analyzer

A command-line tool that analyzes the strength of a password and estimates how long it would take to brute-force.

## Features
- Detects common/weak passwords
- Checks length, uppercase, lowercase, numbers and special characters
- Estimates crack time based on charset size
- Color-coded strength rating (Weak / Fair / Strong / Very Strong)
- Penalizes repeated characters and sequential patterns

## How to run
python password_analyzer.py
